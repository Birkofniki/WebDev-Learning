• The Web wouldn't be possible without HyperText. HyperText is what lets people jump from one web page to another, through links.
  • HyperText is text that contains a link to another page.
  • Web pages are called HyperText documents because they are connected by HyperText links (or hyperlinks)
  • Hyperlinks allow users to [1]MOVE from one web page to another; and also [2]. share INFORMATION across different machines and systems
  • The anchor tag <a> is used to create a hyperlink on a web page. The <a> tag is a container tag.

    • Hyperlinks are used to link from one web page to others. To create a link, you need "href" to add the destination URL.

  • href stands for HyperText reference.

    • E.g Complete the line of code to create a hyperlink

  • <a href="url">Z</a>
  • <a href="url">Text </a>
  • The URL needs to be enclosed in quotes to work without errors, either single or double quotes are okay. If you forget to add href or the quotes, the <a> tag won't create a hyperlink. i.e URLs can be enclosed in matching single (') or double quotes ("). There is no difference as long as they match.

    • What's wrong with this line of code? <a "https://www.sololearn.com">Sololearn</a>

  • The href is missing

    • You can nest hyperlinks inside other text elements. Run the code to see what the web browser will display

  • The content of the anchor tag <a> is the text that you want to display.

    • You can also use hyperlinks to connect pages of the same website. The part of the URL that stays the same for all of a website's pages is the domain name (e.g. sololearn.com).

  • HTML stands for HyperText Markup Language.
    • HTML stands for HyperText Markup Language. The "HyperText" is because web pages have LINKS. "Markup" refers to the SYMBOLS like </>.
