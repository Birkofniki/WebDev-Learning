There are millions of pages on the Web and thousands of web professionals designing and developing web pages every day. In this lesson you'll start applying some of the standards and best practices used in the web industry.

Standards and best practices are needed to make:

  • 1. web pages comptible with different browsers and devices.
  • 2. teamwork and collaboration more effective

  • The <body> container tag is used to group everything that gets displayed on a page when loaded in a browser.
  • Body tags are needed for your page to be compatible with all web browsers.
  • A web page can contain only one body element.
  • All the content elements that you need to display (like paragraphs, headings, buttons and images) need to be inside the <body> container tag.
  • The <body> container tag needs to surround all the elements displayed on the page. When some HTML tags go inside others, this is called nesting. Complete the code for body of the page <body> <p>The content of the page</p> </body>
  • Websites do more than just display content. The <head> container tag is used to include technical information about the page.
  • You can use the head to help increase visibility and traffic from search engines like Google. Title, description and keywords are nested in the head tag. Complete the code to add the head <head> <title>Gardening</title> <meta name="keywords" content="apple, tree"> </head>
  • The head needs to be placed before the body.
  • The information in the head is not displayed on the web page. Only the title is shown, in search engine results (e.g. Google) and in the browser tab.
  • Today, the majority of web professionals always wrap their HTML code in <html> tags. The <html> tag is a container tag. Head and body are nested inside the <html> tags.
  • Indentation is considered a very good practice to improve code readability. Indentation is the spaces at the beginning of lines.
  • Different levels of indentation are used to show nesting.

  • Lesson Takeaways:

    Congratulations! You learned that:

    ⭐ HTML documents can be organized into head and body

    ​⭐ The <body> tag ensures compatibility with all browsers

    ⭐ Indentation helps with readability and therefore collaboration
