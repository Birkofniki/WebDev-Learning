  • text-shadow .............///////////// The text-shadow property allows you to set a shadow for the text. It takes 4 values, like this: CODE PLAYGROUND: CSS h1 { text-shadow: 3px 2px 3px red; } Click to run The first value is the horizontal offset of the shadow. The second value - the vertical offset. The third value is the blur radius: a higher value means the shadow is dispersed more widely. The last one is the color of the shadow.
