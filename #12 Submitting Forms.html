An HTML form is a convenient way to send data to a database hosted in a server. In this lesson, you'll learn to submit form data.

  • A submit button is used to send the data in a form. The submit type of input adds a button to the form.

  • You can use the submit input to send the data in the form to a database hosted in a server.
    • Do you remember what a server is? -A computer that is always connected to the Internet and listening for requests of information (Correct ANSWER) -The device that is being used to access the Web
  • The name attribute is used to reference the data after submitting the form.
  • Only form elements with a name attribute will have their values passed to the database when submitting a form.
